Blog Post Editor

How This Works:

  1. Copy the HTML content from your Google Sheets "Blog Posts" tab
  2. Paste it into the "Post Content" box below
  3. Add a title (or keep the one from Sheets)
  4. Verify the date is correct
  5. Click "Generate Post" - it will download an HTML file to your Mac
  6. Review the downloaded file in your browser
  7. If it looks good, move it to your site's /blog/ folder
  8. Use GitHub Desktop to commit and push
✓ Blog post file generated and downloaded! Check your Downloads folder.
